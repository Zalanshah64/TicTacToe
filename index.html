How to Play

PLAYERS SHOULD NOT SEE THIS TEXT Score: 0

Tic-Tac-Toe, also known as Noughts and Crosses, is a game played by many children around the world. The game is very old, and early versions of it can be found in ancient Egypt and the Roman Empire.

The game starts with a three by three grid of squares, or a pound symbol.

Two players take turns filling in the grid with their respective symbol (X or O). X always starts, and O goes second.

If one player takes a certain square, their oponent will not be able to take that square for the rest of the game.

In order to win, you must line up three consecutive spaces of your symbol either vertically, horizontally, or diagonally.

If every space has been filled in and there has not been a three-in-a-row yet, the game is considered a tie and is over.

When the game ends, the user will have the ability to play again or go to the main menu. Playing again will restart the game, but it will keep the scores. Exiting to the main menu will reset the scoreboard.

Suggestions are a useful tool that allow you to see what the board would look like if you clicked a certain square on that turn.

The settings menu has many useful tools that allow you to change the look and feel of the game. They include:

  • The volume bar, which allows you to change the volume of all sounds
  • The suggestions switch, which allows you to turn on and off suggestions
  • The icon options for each player, which allow you to set the icon representing player one and player two
